HEXATAK

A Hexagonal Grid Connection Puzzle inspired by Tak

HOW TO PLAY

  • To move a stack of stones: Select it by left-clicking, then click an adjacent hex to move it there.
  • Move a single stone to an adjacent hex by clicking the target neighbor (a STEP).
  • Spread multiple stones along a straight line path by clicking, holding, and dragging (a SPREAD).
  • Connect the colored target boundaries to complete the node loop and solve the grid.

SHORTCUTS

  • U Undo the last move
  • Y Redo the last undone move
  • R Reset the current level layout
  • Esc Return to level select / exit screen
  • Space or Click Next level (when won)
